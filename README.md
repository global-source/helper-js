# helper-js
Base helper functions for JavaScript for regular developement uses.

---

#### Count Items [For Object]
To get length of an object.

    ex.
```javascript
     var ob = {'a','b','c','d'};
     ob.length()      // 4

```

#### Generate Range [For Number]
To generate the range of numbers.

    ex.
```javascript
    var n = 5;
    n.range()     // [0,1,2,3,4,5]
```
